## API 

To register/signup a user: 
```POST /api/signup```

To login a user:
`POST /api/login`

To login using Google Sign In:
`POST /api/login/google`

To logout current user:
`POST /api/logout`

To get current user profile data:
`GET /api/user`

To edit current user profile data:
`POST /api/user`

To edit current user profile data:
`POST /api/user`

To upload/edit current user profile image:
`POST /api/user/image`
